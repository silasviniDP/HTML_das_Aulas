## 🎨 HTML_das_Aulas

🧠 Repositório de estudos focado em prática de HTML durante aulas

---

🧩 Sobre

📝 Este projeto reúne arquivos criados ao longo das aulas para praticar os fundamentos da construção de páginas web.

Aqui você vai encontrar exemplos simples, experimentos e exercícios — tudo organizado como parte do processo de aprendizado.
